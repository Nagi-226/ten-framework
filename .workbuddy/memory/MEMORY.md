# TEN Framework 项目分析报告

## 项目基本信息
- **项目名称**: TEN Framework
- **类型**: 开源实时多模态对话AI框架
- **开发者**: Agora（声网）
- **许可证**: Apache License 2.0（有额外限制）
- **主要语言**: Python, Go, TypeScript, C, Rust
- **GitHub仓库**: https://github.com/TEN-framework/ten-framework
- **官方文档**: https://theten.ai/docs

## 核心架构特点
1. **图（Graph）架构**: 基于扩展（Extension）的模块化设计，通过property.json配置图结构
2. **实时多模态**: 支持语音、视频、文本的实时交互，低延迟优化
3. **扩展系统**: 60+扩展库，涵盖ASR、TTS、LLM、RTC、工具等
4. **可视化配置**: 提供TMAN Designer可视化工具（localhost:49483）

## 技术栈分析
- **核心框架**: C/Rust/TypeScript混合实现，高性能运行时
- **AI扩展**: Python为主，支持Deepgram、OpenAI、ElevenLabs、Azure等主流AI服务
- **API服务器**: Go语言实现，RESTful API管理代理生命周期
- **前端界面**: Next.js + React，现代化Web界面
- **部署方案**: Docker容器化，支持Vercel、Netlify等云服务部署

## 项目结构
```
ten-framework/
├── ai_agents/              # AI代理系统（主要开发区域）
│   ├── agents/            # 扩展和示例（60+扩展，20+示例）
│   ├── server/            # Go API服务器
│   └── playground/        # Next.js前端UI
├── core/                  # 核心框架（C/Rust/TypeScript）
├── packages/              # 包管理
├── docs/                  # 多语言文档
└── third_party/           # 第三方依赖
```

## 丰富的示例应用
1. **语音助手**：基础语音对话（STT→LLM→TTS）
2. **实时语音助手**：使用OpenAI Realtime API
3. **视频助手**：支持视觉能力
4. **涂鸦板**：语音/文本生成手绘草图
5. **说话人分离**：实时说话人识别
6. **唇形同步**：支持多个虚拟形象供应商
7. **SIP电话**：支持电话呼叫
8. **ESP32集成**：在硬件开发板上运行
9. **记忆增强**：多种记忆系统集成

## 开发工作流程
1. **环境准备**: Docker + Node.js + Python
2. **配置设置**: .env文件配置API密钥
3. **扩展开发**: Python扩展，继承基类，注册addon
4. **图配置**: property.json定义节点和连接
5. **测试部署**: 本地测试 → Docker打包 → 云部署
